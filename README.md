
# Node.js-Kodecamp-Task2

This repository contains solutions to basic JavaScript array and string tasks.

## Tasks Included

### 1. Flatten a 2D Array
A JavaScript function that converts a two-dimensional array into a single flattened array without using `.flat()`.

#### Example:

Input:
```js
[[1, 2, 3], [4, 5], [6, 7, 8, 9]]
````

Output:

```js
[1, 2, 3, 4, 5, 6, 7, 8, 9]
```

---

### 2. Find Shared Elements Between Three Arrays

A JavaScript function that returns elements shared among three arrays.

#### Example:

Input:

```js
[1, 2, 3, 4]
[3, 4, 5]
[3, 4, 5, 6]
```

Output:

```js
[3, 4]
```

---

### 3. Return Unique Elements

A JavaScript function that returns only unique elements from an array without using `Set` or `filter()`.

#### Example:

Input:

```js
[1, 3, 2, 3, 4, 5, 4, 2, 5, 6]
```

Output:

```js
[1, 2, 3, 4, 5, 6]
```

---

### 4. Convert Kebab Case to Camel Case

A JavaScript function that converts a kebab-case string into camelCase.

#### Example:

Input:

```js
hello-there
```

Output:

```js
helloThere
```
